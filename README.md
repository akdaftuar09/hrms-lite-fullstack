# HRMS Lite

A lightweight full-stack Human Resource Management System built for employee management and attendance tracking.

## Features

- Add employee records
- Delete employee records
- View employee list
- Mark daily attendance
- View attendance records
- Dashboard summary cards
- Attendance filtering by date
- Responsive modern UI
- Success popup notifications

## Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Axios

### Backend
- FastAPI

### Database
- MongoDB Atlas

## Deployment

### Frontend
https://hrms-lite-fullstack-85vm.vercel.app/

### Backend API
https://hrms-lite-backend-zxl8.onrender.com/docs

## Run Locally

### Backend

```bash
cd server
pip install -r requirements.txt
uvicorn app.main:app --reload
```

### Frontend

```bash
cd client
npm install
npm run dev
```
